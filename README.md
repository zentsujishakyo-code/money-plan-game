# マネープランゲーム（ブラウザ版）

香川県子どもの未来応援ネットワークの「マネープランゲーム」を、
ブラウザ（スマホ・パソコン）で遊べるようにしたものです。

サーバーやデータベースは不要で、3つのファイルだけで動きます。

---

## ファイルの中身

| ファイル | 役割 | さわる頻度 |
|---|---|---|
| `index.html` | 画面の見た目 | ときどき |
| `game.js` | ゲームの動き（プログラム本体） | あまり |
| `config.js` | **カードの中身・金額・確率などの設定** | **ここをよく調整する** |

調整したいことのほとんどは `config.js` の数字や文章を書きかえるだけでできます。
プログラム（`game.js`）は読めなくても大丈夫です。

---

## まず手元で動かしてみる（公開の前に）

1. 3つのファイル（`index.html` `config.js` `game.js`）を**同じフォルダ**に置く
2. `index.html` をダブルクリックしてブラウザで開く

これだけで動きます。スマホで見たいときは、下の「GitHub Pagesで公開」に進みます。

---

## GitHub Pages で公開する手順

> スマホの実機で開けるようにするには、インターネット上に置く必要があります。
> GitHub Pages なら無料で、URL（`https://○○.github.io/△△/`）が手に入ります。

### 初回だけ：リポジトリを作る
1. GitHub にログイン → 右上の「＋」→ **New repository**
2. リポジトリ名を入力（例：`money-plan-game`）
3. **Public** を選ぶ（Pages は Public で使うのが簡単）
4. **Create repository**

### ファイルを置く
1. 作ったリポジトリの画面で **Add file → Upload files**
2. `index.html` `config.js` `game.js` の3つをドラッグして上げる
3. 下の **Commit changes** を押す

### Pages を有効にする
1. リポジトリの **Settings** → 左メニューの **Pages**
2. 「Build and deployment」の Source を **Deploy from a branch**
3. Branch を **main**、フォルダは **/(root)** を選んで **Save**
4. 1〜2分待つと、ページ上部に公開URLが出る
   （`https://あなたのID.github.io/money-plan-game/`）

このURLをスマホで開けば、実機で遊べます。
QRコードにして配ると、教室で配布しやすくなります。

### 直したいとき（2回目以降）
- `config.js` などを直して、同じ場所に Upload して Commit するだけ
- 1〜2分でURLの中身が新しくなります

## よくある調整（config.js）

`config.js` をテキストエディタで開いて、次の場所を書きかえます。

- **金額をかえる** … `cost`（払う額）`gain`（もらう額）`win`（福引き当たり額）の数字
- **文章をかえる** … `text` `subtitle` などの文字
- **お給料をかえる** … 先頭の `SALARY`
- **福引きの当たりやすさ** … `FUKUBIKI_WIN_RATE`（`0.4` なら40%）
- **支払日カードのばらけ方** … `SPREAD_PAYDAY`（`true`＝固定費が片寄らない）

数字を直したら保存して、また Upload すれば反映されます。
